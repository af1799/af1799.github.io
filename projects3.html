Scute-Shoot

Scute-Shoot is a casual adventure game from the 2025 Cornell DGA Summer Project. I contributed to player programming, UI programming, and UI art, and authored the game concept document used to pitch the project. I implemented player character mechanics, designed UI elements, and helped plan the audio system architecture for scalable sound integration. Check it out on itch.io!
